The Vehicle Rental System is a console-based Java application designed to manage vehicle rental operations efficiently.
It supports three roles: Admin, Staff, and Customer, each with specific functionalities and access permissions. 
The system allows vehicle management, rental registration, booking, and transaction tracking.
It is developed using Object-Oriented Programming concepts and uses ArrayList for data storage. 
The application ensures role-based login and smooth rental management through a menu-driven interface.
